Authentication

Base path: /api/auth · Rate limited: 10 req / 15 min per IP

POST /api/auth/register Create a new user account ▼
Roles
Public
Body Parameters
Field Type Required Rules
name string yes min 2 chars
email string yes valid email
password string yes min 8 chars, 1 uppercase, 1 digit
role enum opt VIEWER | ANALYST | ADMIN
⚠️
The role field is self-assignable — a known limitation. In production, restrict role assignment to admins.
